Перед визитом к клиенту — обязательно проверить

Эти пункты нужно подготовить ДО встречи с клиентом. Не импровизируй на месте — всё должно быть готово заранее.

  • Zoom установлен и Remote Control включён в настройках безопасности вашего аккаунта Zoom
  • У вас есть доступ к репозиторию VIVI на GitHub (проверь что можешь его форкнуть/клонировать)
  • Тестовый офис работает — убедись что всё актуально до выезда к клиенту
  • Zoom-созвон запланирован — клиент знает время и уже в нём
  • Ссылка на репозиторий VIVI готова, чтобы скинуть клиенту в чат
Комментарий для рабочего: Представься клиенту, кратко объясни что сейчас будет происходить: «Мы вместе за 3–4 часа настроим ваш персональный цифровой офис. Я буду вести вас шаг за шагом — вам ничего не нужно знать заранее.» Убедись что клиент комфортно расположился, есть доступ к интернету и зарядка ноутбука.

GitHub не приходит письмо с кодом подтверждения

Часть A.2 — Оплата Claude Pro из России (выбери один способ)

✅ Способ 1 — Ozon Bank (самый простой)

Оплата рублями прямо из приложения Ozon. Не нужны крипта и иностранные карты.

Шаги:
1. Открой приложение Ozon на телефоне клиента
2. Перейди в раздел «Услуги» → «Иностранные сервисы»
3. Найди и выбери Claude
4. Введи email аккаунта Claude → выбери период (1 месяц)
5. Оплати рублями через Ozon Pay
6. На email придёт код активации → активируй на claude.ai
7. В профиле Claude появится «Pro Plan» ✅

💳 Способ 2 — Посредник (быстро, за рубли)

Сервис-посредник покупает подписку за тебя. Стоимость ~2 300–2 500₽ (включая комиссию ~450₽).

Шаги:
1. Напиши в Telegram посреднику (уточни актуальный у куратора VIVI)
2. Укажи: email аккаунта Claude + план Pro
3. Оплати через СБП рублями
4. Менеджер активирует подписку на аккаунт клиента
5. Проверь: в профиле Claude должно быть «Pro Plan» ✅

🌍 Способ 3 — Иностранная виртуальная карта

Подходит если клиент хочет карту на будущее (для оплаты других сервисов тоже).

Шаги:
1. Зарегистрируйся на Wise.com или Revolut.com
2. Пройди верификацию (паспорт + селфи, ~10–15 мин)
3. Пополни счёт переводом с российского банка (через SWIFT или агента)
4. Создай виртуальную карту → скопируй номер, CVV, срок
5. На claude.ai → «Upgrade to Pro» → введи данные виртуальной карты
6. VPN должен быть включён на USA при оплате ⚠️
7. «Pro Plan» активирован ✅

Карта отклоняется при оплате на claude.ai
Российские карты не работают — используй один из трёх способов выше. Если пробуешь Wise/Revolut — убедись что VPN включён на США в момент оплаты, иначе Stripe заблокирует транзакцию по IP.
Ozon не показывает Claude в «Иностранных сервисах»
Обнови приложение Ozon до последней версии. Если Claude всё равно нет — попробуй способ 2 (посредник). Наличие сервисов в Ozon периодически меняется.
Wise/Revolut не верифицирует клиента
Попробуй Revolut вместо Wise (или наоборот) — у них разные требования к документам. Если оба не работают — переходи к способу 2 (посредник через Telegram), это самый надёжный вариант.
Код активации от Ozon не работает
Убедись что вводишь код в разделе «Redeem a code» на claude.ai (не в поле оплаты картой). VPN должен быть включён на США в момент активации.

Установка Claude Code на Windows
  1. Открой PowerShell от имени администратора: нажми Win → введи «PowerShell» → правый клик → «Запустить от имени администратора»
  2. Вставь команду и нажми Enter:
    irm https://claude.ai/install.ps1 | iex
  3. Жди завершения установки (1–2 минуты) — ты увидишь «Claude Code installed successfully»
  4. Закрой PowerShell и открой его заново (обычный, не от администратора)
  5. Введи команду:
    claude
  6. Откроется запрос на авторизацию → нажми Enter → откроется браузер → войди в аккаунт Claude клиента → нажми «Allow»
  7. Claude Code авторизован в терминале ✅
  8. Теперь подключи в Cursor: открой Cursor → нажми Ctrl+Shift+X (Extensions) → введи «Claude Code» → установи расширение от Anthropic → нажми «Sign in»
  9. Откроется браузер → войди в тот же аккаунт Claude → «Allow»
  10. В Cursor появится значок Claude Code в боковой панели ✅
  11. Тест: открой папку с репозиторием VIVI в Cursor → нажми на значок Claude Code → напиши «привет» → убедись что офис ответил ✅
Установка Claude Code на Mac
  1. Открой Terminal: нажми Cmd+Пробел → введи «Terminal» → Enter
  2. Вставь команду и нажми Enter:
    curl -fsSL https://claude.ai/install.sh | bash
  3. Жди завершения (1–2 минуты) — увидишь «Claude Code installed successfully»
  4. Закрой Terminal и открой заново
  5. Введи команду:
    claude
  6. Откроется запрос авторизации → нажми Enter → браузер откроется → войди в аккаунт Claude → нажми «Allow»
  7. Claude Code авторизован ✅
  8. Открой Cursor → нажми Cmd+Shift+X (Extensions) → найди «Claude Code» от Anthropic → установи
  9. Нажми «Sign in» → войди в тот же Claude-аккаунт
  10. В боковой панели Cursor появился значок Claude Code ✅
  11. Тест: открой папку VIVI в Cursor → Claude Code → напиши «привет» → офис ответил ✅
Команда claude не находится после установки
На Windows: закрой и снова открой PowerShell. Если не помогает — выйди из системы и зайди снова. На Mac: закрой Terminal → открой заново → попробуй команду which claude. Если не находит — переустанови командой curl.
Claude Code не видит репозиторий VIVI
В Cursor нажми «File → Open Folder» и выбери папку с репозиторием. Claude Code работает только внутри открытого проекта. Убедись что папка VIVI клонирована (git clone) или скопирована на компьютер.
Расширение Claude Code не появляется в Cursor
Попробуй установить вручную: найди папку с Claude Code (Windows: C:\Users\[имя]\.claude, Mac: ~/.claude) → найди файл .vsix → перетащи в окно Extensions в Cursor.
При оплате Claude Pro карта отклоняется
Убедись что VPN включён и показывает США. Российские карты обычно не работают — используй иностранную карту или попроси клиента завести карту в иностранном банке/виртуальную карту (Wise, Revolut).
